Informations

🌿 Comment fonctionne MemoFlash ?

MemoFlash est une application d'apprentissage gratuite, open source et sans compte.

  • Tes decks restent sur ton appareil. Tout est stocké dans ton navigateur (localStorage).
  • Pas de compte nécessaire. Ouvre l'app et commence directement.
  • Partage par code. Publie un deck pour obtenir un code à 6 caractères à partager.
  • Import Quizlet. Copie-colle tes sets Quizlet directement dans Réglages.
  • Sauvegarde. Exporte tes decks en .json pour ne jamais les perdre.

❓ FAQ

Est-ce vraiment gratuit ? +
Oui, totalement. Pas d'abonnement, pas de fonctionnalités cachées. Le code source est ouvert sur GitHub sous licence MIT.
Mes données sont-elles sécurisées ? +
Tes decks locaux sont uniquement dans ton navigateur — personne d'autre ne peut y accéder. Seuls les decks que tu publies volontairement sont envoyés sur le serveur.
Comment ne pas perdre mes decks ? +
Réglages → Exporter. Tu obtiens un fichier .json à garder. Si tu vides ton cache ou changes de navigateur, importe ce fichier pour tout retrouver.
Comment importer depuis Quizlet ? +
Dans Quizlet : ouvre ton set → "..." → "Exporter" → séparateur Tabulation → "Copier le texte". Puis dans MemoFlash : Réglages → "Import Quizlet" → colle le texte.
Le site marche-t-il sans internet ? +
Oui pour tout ce qui est local (créer, réviser, quiz). Non pour rechercher ou partager des decks communautaires, qui nécessitent le serveur.
Comment fonctionne la révision espacée ? +
On utilise l'algorithme SM-2 (le même qu'Anki). Les cartes que tu trouves faciles reviennent moins souvent, les difficiles plus souvent. Chaque session optimise ton temps de révision.
